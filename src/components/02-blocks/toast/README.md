# Toast

## Block

`.toast`

## Modifiers

| Class            | Description                                                  |
| ---------------- | ------------------------------------------------------------ |
| `.toast--danger` | Styles a `.toast` with the danger color to inform the user of system errors |

## States


| Class               | Description                                        |
| ------------------- | -------------------------------------------------- |
| `.toast.is-visible` | Styles the `.toast` to fade in and move into place |
