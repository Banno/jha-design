# Thumbnail

## Block

`.thumbnail`

## Modifiers

| Class            | Description             |
| ---------------- | ----------------------- |
| `.thumbnail--xs` | Sizes thumbnail to 24px |
| `.thumbnail--sm` | Sizes thumbnail to 36px |
| `.thumbnail--md` | Sizes thumbnail to 48px |
| `.thumbnail--lg` | Sizes thumbnail to 72px |
| `.thumbnail--xl` | Sizes thumbnail to 96px |
