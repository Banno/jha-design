# Tab

## Block

`.tabs`

## Elements

| Class       | Requirement | Description       |
| ----------- | ----------- | ----------------- |
| `.tab_item` | Required    | Creates a new tab |

## States

| Class                 | Description                                                  |
| --------------------- | ------------------------------------------------------------ |
| `.tab_item.is-active` | Styles a .`tab_item` to indicate which one is currently active |
